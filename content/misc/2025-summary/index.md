时间过得很快，我每年这个时候都会写一篇流水账，来记录过去一年的生活和工作。希望未来的自己还能记得这一年都发生了什么。

## 年初的 Flag 完成情况

回看自己年初写下的 Flag，有些实现了，有些做了但不是很好，还有一些没有做到。

1. 去其他欧洲国家旅行，争取看一场喜欢的球队比赛。

   完成了一半，去了奥地利，但没去过像西班牙、意大利这样的足球强国和看比赛。

2. 争取获得 Azure 认证。

    随着 AI 快速发展，我觉得这个目标不再那么重要了，直接放弃了。
  
3. 争取参与 DevOps 或 Python 大会。

  四月份去了 PyCon LT。写过三篇大会的文章：[PyCon LT 2025 第一天](https://shenxianpeng.github.io/misc/pycon-lt-d1/)、[PyCon LT 2025 第二天](https://shenxianpeng.github.io/misc/pycon-lt-d2/)、[PyCon LT 2025 第三天](https://shenxianpeng.github.io/misc/pycon-lt-d3/)。

4. 争取加入 PyPA 或 Python GitHub Organization。

  今年创建了一个新的开源项目 [Explain Error Plugin](https://plugins.jenkins.io/explain-error/)，所以加入了 Jenkins GitHub Organization。
  加入 PyPA 和 Python GitHub Organization 的目标似乎并没有那么重要，但我还是希望未来能有机会加入。

5. 争取别长胖，保持运动。

  这一年我没怎么长胖，但运动量也不大。2026 希望能做的更好一些。

总结来说，虽然没有全部完成，但也算是不错的成绩了。

## 回顾 2025

其实我已经不记得 2025 年发生了什么了，所以我翻看了日历和照片，回忆了一下这一年的点滴。

* 一月：在维尔纽斯过的新年，月底了准备回国探亲。
* 二月：回老家过年，但因为没有得到充分休息，感冒了好几天，后面得了水痘。随后就是边工作然后抽空陪家人、直到月底我的水痘好了才算才开始见到一些朋友。
* 三月：开始间断的休假，去海边、去圣亚、动物园等地方陪孩子和家人。
* 四月：又回到了维尔纽斯，每次都是带着不舍的心情离开，只有到了才慢慢回到这里的生活和工作。随后参加了 PyCon LT 2025、带孩子去看了一场篮球比赛。
* 五月：自驾去了考纳斯，在那里给孩子过了三岁的生日，参观了考纳斯动物园、考纳斯老城等地标性建筑。月底去看了维尔纽斯粉红色汤节。
* 六月：奥地利之行，去了沃尔夫冈（很美）、维也纳等地。月底第一次去了同事家做客。
* 七月：坐火车去了克莱佩达，参观了海洋博物馆，主要是想去海边玩玩沙子和水，可惜那几天的天气不太好。
* 八月：逛逛老城，准备迎接孩子幼儿园开学。
* 九月：孩子顺利进入幼儿园，虽然哭过两次，但逐渐适应的挺好。带孩子参加了维尔纽斯的首都节活动。
* 十月：按部就班的工作和生活，周末多出门走走，到下个月就很难看到这么好的天气了。
* 十一月：还是一样，本来有计划去波兰，但因为工作太忙就取消了，准备今年有机会再看。
* 十二月：随着休假的临近，我也更加忙碌。参加了维尔纽斯的圣诞集市，孩子幼儿园的圣诞活动，就踏上了回国的旅程。

这些点滴构成了2025年珍贵的记忆。

## 展望 2026

2026 年，我不想立太多的 Flag，只想做好以下几件事：

1. 去其他欧洲国家旅行，~~争取~~看一场~~喜欢的~~足球比赛。
2. ~~争取~~参与 ~~DevOps 或~~ Python 大会。
3. ~~争取~~别长胖，保持运动。
4. 继续坚持写作和开源，尤其是我新开始的《攻城狮周刊》这个周更项目。
5. 更多的关注 AI 相关的技术和应用，深度将它融入到我的工作和学习中。

最后，祝大家新年快乐，万事如意！
