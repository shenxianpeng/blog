时间过得很快，2025 年已经过去。如果不记录下在这一年里发生的事情，过不了多久就很难回想起这一年都发生了什么。按照惯例，这篇年终总结如期而至。

回看自己年初写下的 Flag，有些实现了，有些做了但不是很好，还有一些没有做到。

1. 去其他欧洲国家旅行，争取看一场喜欢的球队比赛。

   完成了一半，去了奥地利，但没看比赛。

2. 争取获得 Azure 认证。

    随着 AI 以及工作内容的调整，这个目标被放弃了。
3. 争取参与 DevOps 或 Python 大会。

  四月份去了 PyCon LT。写过三篇大会的文章：[PyCon LT 2025 回顾](https://shenxianpeng.github.io/2025/04/pycon-lt-2025-review/)、[PyCon LT 2025 会议笔记（上）](https://shenxianpeng.github.io/2025/04/pycon-lt-2025-notes-part1/)、[PyCon LT 2025 会议笔记（下）](https://shenxianpeng.github.io/2025/04/pycon-lt-2025-notes-part2/)。

4. 争取加入 PyPA 或 Python GitHub Organization。

  加入了 Jenkins Organization，但还没加入 PyPA 或 Python GitHub Organization。

5. 争取别长胖，保持运动。

  还好没有长胖。希望 2026 年也能减重。

总结来说，2025 的目标的大体上完成的还不错。

## 回顾 2025

回顾这一年，这一年的关键词就是“欧洲生活”。

* 一月：在维尔纽斯过的新年，参观了老城、格迪米纳斯塔、总统府、圣安娜教堂等地标性建筑。s
* 二月：回国探亲，陪孩子过了一个生日。孩子烫伤的伤口终于痊愈了，全家人都松了一口气。
* 三月：继续在国内探亲，带爸妈去了丹东，看了鸭绿江大桥和抗美援朝纪念馆。
* 四月：回到了维尔纽斯，参加了 PyCon LT 2025。
* 五月：去了考纳斯，参观了考纳斯老城、考纳斯城堡、圣彼得保罗教堂等地标性建筑。
* 六月：奥地利维也纳之旅，参观了美泉宫、霍夫堡宫、维也纳国家歌剧院等地标性建筑。
* 七月：克莱佩达之旅，参观了克莱佩达老城、雕塑公园、海洋博物馆等地标性建筑。
* 八月：逛逛老城，准备迎接孩子幼儿园开学。
* 九月：孩子顺利进入幼儿园，适应良好。
* 十月：参加了维尔纽斯的万圣节活动，带孩子去了南瓜节。
* 十一月：参加了维尔纽斯的感恩节活动，和家人一起庆祝。
* 十二月：准备迎接新年，参加了维尔纽斯的圣诞市场。回国过圣诞节和新年。

这些点滴构成了2025年珍贵的记忆。

### 享受每一天的工作

上半年，我一边准备交接，一边完成本职工作，始终不遗余力地践行 DevOps 最佳实践。尽管结果尚不可知。

但我真的很享受在岗的每一天，包括咖啡、午休散步、听喜欢的内容、偶尔跑步，以及每周二、四的羽毛球时光。

### 业余时间，坚持开源和写作

和去年一样，业余时间我还做着相同的事情：

1. 开源项目
    * 继续维护着 cpp-linter，已有上千用户，包括微软、Jupyter、Linux Foundation 等知名项目。
    * 对 commit-check 做了重大的重构和优化，用户数稳步增长。
    * gitstats：这是我今年新纳入维护的一个开源项目，希望能帮助更多人。
    * explain-error Jenkins plugin 是我今年新创建的一个开源项目，希望能帮助 Jenkins 用户更好地理解错误信息。
    * gnuplot-wheel 也是我今年新创建的一个开源项目，
    * jenkinsfilelint 这是我今年新创建的一个开源项目，希望能帮助 Jenkins 用户更好地编写 Jenkinsfile。
    * clang-tools-wheels 这是我今年新创建的一个开源项目，希望能帮助 C/C++ 开发者更好地使用 clang 工具链。
2. 写作 — 全年更新 41（去年27）篇博客、60（去年41） 篇公众号文章，大幅超出预期。

## 展望 2026

2026 年，我不想报太多的 Flag，毕竟生活和工作都充满了不确定性。

如果有什么愿望，那就是：

1. 祝愿家人身体健康。
2. 去其他欧洲国家旅行，~~争取~~看一场~~喜欢的~~足球比赛。
3. ~~争取~~参与 ~~DevOps 或~~ Python 大会。
4. ~~争取加入 PyPA 或 Python GitHub Organization。~~
5. ~~争取~~别长胖，保持运动。
6. 继续坚持写作和开源，尤其是在 AI 领域。
