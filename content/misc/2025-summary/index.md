时间过得很快，每到年末，我都会写一篇流水账，记录过去一年的生活和工作。希望未来的自己还能记得这一年都发生了什么。

## 年初的 Flag 完成情况

回顾年初写下的 Flag，有些实现了，有些完成了但还不够理想，也有一些没做成。

1. 去其他欧洲国家旅行，看一场足球比赛

   完成了一半，去了奥地利，但没去像西班牙、意大利这样的足球强国，也没看成比赛。希望今年能实现。

2. 获得 Azure 认证

    随着 AI 快速发展，这个目标看起来不再那么重要，就暂时放下了。
  
3. 参与 DevOps 或 Python 大会

    四月份去了 PyCon LT，写了三篇大会文章：[第一天](https://shenxianpeng.github.io/misc/pycon-lt-d1/)、[第二天](https://shenxianpeng.github.io/misc/pycon-lt-d2/)、[第三天](https://shenxianpeng.github.io/misc/pycon-lt-d3/)。

4. 加入 PyPA 或 Python GitHub Organization

    今年创建了新开源项目 [Explain Error Plugin](https://plugins.jenkins.io/explain-error/)，因此加入了 Jenkins GitHub Organization。加入 PyPA 和 Python GitHub Organization 的目标暂时没实现，但希望未来有机会加入。

5. 别长胖，保持运动

    今年没怎么长胖，但运动量也不大。2026 希望能做得更好一些。

总结来说，虽然没有全部完成，但也算是不错的成绩了。

## 回顾 2025

其实我已经不记得 2025 年发生了什么了，所以我翻看了日历和照片，回忆了一下这一年的点滴。

* 一月：在维尔纽斯过的新年，月底了准备回国探亲。
* 二月：回老家过年，但因为没有得到充分休息，感冒了好几天，后面得了水痘。随后边工作边陪家人，水痘痊愈后才开始见朋友。
* 三月：开始间断的休假，去海边、去圣亚、动物园等地方陪孩子和家人。
* 四月：又回到了维尔纽斯，每次离开都带着不舍，回到维尔纽斯才能慢慢回到生活和工作节奏。随后参加了 PyCon LT、带孩子去看了一场篮球比赛。
* 五月：自驾去了考纳斯，在那里给孩子过了三岁的生日，参观了考纳斯动物园、考纳斯老城等地标性建筑。月底去看了维尔纽斯粉红色汤节。
* 六月：奥地利之行，去了沃尔夫冈（湖光山色非常美）、维也纳等地。。月底第一次去了同事家做客。
* 七月：坐火车去了克莱佩达，参观了海洋博物馆，主要是想去海边玩玩沙子和水，可惜那几天的天气不太好。
* 八月：逛逛老城，准备迎接孩子幼儿园开学。
* 九月：孩子顺利进入幼儿园，虽然哭过两次，但逐渐适应的挺好。带孩子参加了维尔纽斯的首都节活动。
* 十月：按部就班的工作和生活，周末多出门走走，到下个月就很难看到这么好的天气了。
* 十一月：还是一样，本来有计划去波兰，但因为工作太忙就取消了，准备今年有机会再看。
* 十二月：随着休假的临近，我也更加忙碌。参加了维尔纽斯的圣诞集市，孩子幼儿园的圣诞活动，就踏上了回国的旅程。

这些点滴构成了2025年珍贵的记忆。

## 展望 2026

2026 年，我不打算立太多 Flag，只希望做好以下几件事：

1. 保持运动，家人和自己都健康。
1. 多陪伴家人，多出去走走，多和别人交流。
4. 坚持写作和开源，尤其是新开始的《攻城狮周刊》周更项目。
5. 更多关注 AI 技术和应用，把它更深地融入工作和学习。

如果想给未来 **1000 天后的自己** 立一个目标，那大概是：

* 开发出属于自己的“第二条腿”，走出更多可能。
* 工作和生活更加灵活自由，并始终保持热情。

希望未来的自己，看到这些文字时，仍能微笑着回忆这一年。

也祝你在新的一年里，身体健康，工作顺利，生活愉快！
